## 2. 當前研究框架摘要
目前研究以 **OpenClaw GitHub 首次公開發布日** 作為單一研究錨點，並以公開財報與公開市場資料為核心。

- **研究主問題**：哪些 SaaS 企業在事件後呈現較高衝擊程度，且這些企業具備哪些共同特徵。
- **主分析方法**：單一事件窗市場反應分析 + 集群分析。
- **主資料來源**：10-Q、10-K、earnings release、公開股價資料、公開法說文本。
- **研究定位**：描述型與關聯型分析，不主張嚴格因果識別。

## 5. 目前主要風險
1. **事件風險**：GitHub 首發日可能是技術公開節點，但未必是市場明確認知事件。
2. **資料風險**：公開財報欄位揭露不完全一致，可能壓縮有效樣本。
3. **方法風險**：若樣本過少或缺值過多，集群分析結果可能不穩定。
4. **敘事風險**：報告需避免使用因果措辭，並清楚區分市場反應與基本面變化。
